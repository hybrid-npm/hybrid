# Hybrid Agent - Template

This is a template agent generated by `hybrid init`.

## Quick Start

```bash
# Install dependencies
npm install

# Run locally
npm run dev

# Build for production
npm run build

# Deploy to Fly.io
fly deploy
```

## Project Structure

```
.
├── package.json          # Agent dependencies
├── start.sh             # Startup script
├── .env.example         # Environment variables template
├── AGENTS.md           # Build/deploy commands (this file)
├── SOUL.md             # Agent identity and personality
└── README.md           # Hybrid README and roadmap
```

## Commands

- `npm run dev` - Run server in development mode
- `npm run build` - Build for production
- `fly deploy` - Deploy to Fly.io

## Dependencies

This agent depends on:
- `hybrid/agent` - Core agent runtime
- `hybrid/gateway` - Webhook gateway (optional)

See `.env.example` for required variables.
