---
title: HTTP API
description: API reference for the Hybrid agent server endpoints
---

The Hybrid agent server exposes two HTTP endpoints via [Hono](https://hono.dev).

## Health check

```
GET /health
```

Returns the server status.

**Response:**

```json
{ "ok": true, "service": "agent-server" }
```

## Agent endpoint

```
POST /api/agent
```

Sends a conversation to the Claude Agent SDK and streams the response back as Server-Sent Events.

### Request body

```typescript
interface ContainerRequest {
  messages: UIMessageLite[]
  systemPrompt: string
  temperature: number
}

interface UIMessageLite {
  id: string
  role: "user" | "assistant" | "system" | "data"
  content: string
  createdAt?: string
}
```

| Field | Type | Description |
|-------|------|-------------|
| `messages` | `UIMessageLite[]` | Conversation history. The last message is sent to the SDK as the prompt. |
| `systemPrompt` | `string` | System prompt merged with the server's `AGENT.md` file. |
| `temperature` | `number` | Sampling temperature passed to the Claude model. |

### Example request

```bash
curl -N http://localhost:4100/api/agent \
  -H "Content-Type: application/json" \
  -d '{
    "messages": [
      { "id": "1", "role": "user", "content": "What can you help me with?" }
    ],
    "systemPrompt": "You are a helpful assistant.",
    "temperature": 0.7
  }'
```

### Response

The response is an SSE stream with `Content-Type: text/event-stream`. See [SSE Protocol](/docs/server/sse) for the full event format.

### Errors

On failure, the endpoint returns a JSON error instead of an SSE stream:

```json
{ "error": "error message here" }
```

**Status codes:**

| Code | Meaning |
|------|---------|
| `200` | Success -- SSE stream begins |
| `500` | Server error -- JSON error body |

### Abort handling

The server respects client disconnection. If the client closes the connection, the SSE stream is cancelled and the underlying Claude Agent SDK query is aborted via `AbortController`.
