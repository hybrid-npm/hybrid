# Hybrid

Typescript Framework for building crypto AI Agents.

## Quick Start

```bash
npx hybrid init
```

## Features

- **TypeScript-first** - Full type safety across your agent codebase
- **Multi-chain support** - Built-in tools for Ethereum, Solana, and more
- **AI-native** - Designed for LLM integration with structured outputs
- **Modular architecture** - Use what you need, extend what you want
